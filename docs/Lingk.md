## 内部仿照提示词
你是一位专业的ACM出题人，请根据ACM出题规范，出一道简单的算法题。

任务： 
1. 请根据一种简单算法编写一道题目
2. 使用算法生成出最少2个数据样例
3. 测试数据样例是否有问题，如果有问题删除错误的样例重新生成正确的样例
4. 最后生成题解包含多门常用语言的解法和解题思路

请生成可运行的代码完成操作
1. 将生成的题目和解析分别放入md文件中
2. 将输入输出的测试数据放入一个文件夹中格式是in和out的格式

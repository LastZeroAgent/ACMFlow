# 寻找两个有序数组的中位数
 ## 题目描述
给定两个大小分别为 m 和 n 的**非递减排序**整数数组 `nums1` 和 `nums2`，请你找出这两个数组合并后的中位数。如果合并后的数组长度为偶数，则中位数是中间两个数的平均值。你可以假设 `nums1` 和 `nums2` 都不为空。

## 输入格式
第一行包含两个整数 m 和 n，分别表示数组 `nums1` 和 `nums2` 的长度。
第二行包含 m 个整数，表示数组 `nums1` 的元素。
第三行包含 n 个整数，表示数组 `nums2` 的元素。

## 输出格式
输出一个浮点数，表示合并后数组的中位数，保留一位小数。

## 样例

### 样例 1

**输入：**
```
2 1<br>1 3<br>2
```

**输出：**
```
2.0
```

### 样例 2

**输入：**
```
2 2<br>1 2<br>3 4
```

**输出：**
```
2.5
```

## 数据范围
1. 本题的核心在于高效地找到两个有序数组的中位数，无需实际合并数组。
2. 注意处理总长度为偶数时的平均值计算。
3. 输入数组已经是非递减排序，无需额外排序。
4. 需要考虑不同长度数组的组合情况。

# 回文数判定
 ## 题目描述
给定一个正整数 `n`，你的任务是判断该数是否为回文数。回文数是指正序（从左向右读）和倒序（从右向左读）都相同的整数。例如，121 是回文数，而 123 不是。

## 输入格式
输入仅包含一个正整数 `n`（1 ≤ n ≤ 10^18）。

## 输出格式
如果 `n` 是回文数，输出 "YES"；否则输出 "NO"。

## 样例

### 样例 1

**输入：**
```
121
```

**输出：**
```
YES
```

### 样例 2

**输入：**
```
123
```

**输出：**
```
NO
```

## 数据范围
1. 不建议直接将数字转为字符串操作，但允许使用此方法简化逻辑。
2. 注意处理大数时的计算效率问题。
3. 特殊边界情况：单个数字一定是回文数。

# 数字各位之和 
## 题目描述
给定一个整数n（可能为负数），请计算该整数各位数字的绝对值之和。例如，-456的各位数字绝对值之和为4+5+6=15。


## 输入格式
输入仅一行，包含一个整数n（-1e9 ≤ n ≤ 1e9）。

## 输出格式
输出一个整数，表示各位数字的绝对值之和。

## 样例

### 样例 1

**输入：**
```
123
```

**输出：**
```
6
```

### 样例 2

**输入：**
```
-456
```

**输出：**
```
15
```

## 数据范围
1. 注意处理负数的情况，需先取绝对值再计算
2. 单个数字直接返回该数字本身
3. 对于0的处理应返回0
4. 建议使用字符串转换或数学运算两种方法实现

# 最大连续子数组和
 ## 题目描述
[详细的题目描述]  
给定一个长度为 `n` 的整数数组 `nums`，请你找出其中具有最大和的**连续子数组**，并返回该最大和。若存在多个解，只需返回任意一个即可。  
**注意**：子数组至少包含一个元素。当数组全为负数时，结果应为数组中的最大单元素值。

## 样例

### 样例 1

**输入：**
```
<br>5<br>-2 1 -3 4 -1 2 4 -5 3
```

**输出：**
```
8
```

### 样例 2

**输入：**
```
<br>3<br>-1 -2 -3
```

**输出：**
```
-1
```

## 数据范围
- 时间限制：1秒
- 内存限制：256MB
- 1 ≤ n ≤ 1e5

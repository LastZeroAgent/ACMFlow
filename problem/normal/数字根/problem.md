# 数字根
 ## 题目描述
给定一个非负整数 `n`，反复将其各位数字相加，直到结果变成一个一位数为止。这个过程称为求数字根。例如，对于 `n = 38`，计算过程为 `3 + 8 = 11` → `1 + 1 = 2`，因此数字根为 `2`。请编写程序计算任意非负整数的数字根。

## 输入格式
输入仅包含一个非负整数 `n`（0 ≤ n ≤ 10^18）。

## 输出格式
输出一个整数，表示 `n` 的数字根。

## 样例

### 样例 1

**输入：**
```
38
```

**输出：**
```
2
```

### 样例 2

**输入：**
```
0
```

**输出：**
```
0
```

## 数据范围
1. 注意处理 `n = 0` 的特殊情况。
2. 可通过数学性质优化：若 `n % 9 == 0` 且 `n != 0`，则数字根为 `9`；否则为 `n % 9`。
3. 避免直接模拟逐位相加导致的超时问题（尤其对极大数）。

# 最大公约数 (GCD)
 ## 题目描述
[详细的题目描述]  
给定两个正整数 `a` 和 `b`，请计算它们的最大公约数（Greatest Common Divisor, GCD）。最大公约数是指能同时整除 `a` 和 `b` 的最大正整数。

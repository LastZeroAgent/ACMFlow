# 最大公约数计算器
 ## 题目描述
给定两个正整数 a 和 b，请计算它们的最大公约数（Greatest Common Divisor, GCD）。最大公约数是指能够同时整除这两个数的最大正整数。本题要求使用高效的欧几里得算法进行求解。

## 输入格式
输入包含一行，两个正整数 a 和 b，以空格分隔。其中 1 ≤ a, b ≤ 10^9。

## 输出格式
输出一个整数，表示 a 和 b 的最大公约数。

## 样例

### 样例 1

**输入：**
```
6 8
```

**输出：**
```
2
```

### 样例 2

**输入：**
```
17 23
```

**输出：**
```
1
```

## 数据范围
1. 推荐使用欧几里得算法（辗转相除法），该算法的时间复杂度为 O(log(min(a,b)))。
2. 注意处理大数运算时的溢出问题，建议使用长整型变量。
3. 当其中一个数为0时，另一个数即为最大公约数（但本题保证输入均为正整数）。

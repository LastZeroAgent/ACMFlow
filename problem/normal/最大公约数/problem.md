# 最大公约数
 ## 题目描述
给定两个整数 `a` 和 `b`，请计算它们的最大公约数（Greatest Common Divisor, GCD）。最大公约数是指能同时整除 `a` 和 `b` 的最大正整数。本题保证输入的整数不全为零。

## 输入格式
输入仅一行，包含两个整数 `a` 和 `b`，中间用空格分隔。

## 输出格式
输出一个整数，表示 `a` 和 `b` 的最大公约数。

## 样例

### 样例 1

**输入：**
```
6 8
```

**输出：**
```
2
```

### 样例 2

**输入：**
```
17 19
```

**输出：**
```
1
```

## 数据范围
1. 使用欧几里得算法（辗转相除法）可高效求解。
2. 注意处理负数输入，需先取绝对值再计算。
3. 当其中一个数为0时，另一个数即为最大公约数。
4. 算法时间复杂度为 O(log(min(|a|, |b|)))。

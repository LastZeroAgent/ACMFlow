# 回文数判断
## 题目描述
判断一个整数是否为回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。注意：所有负数都不是回文数，因为负号的存在使得数字不对称；对于个位数（0～9），它们都是回文数；末尾为 0 的正整数（如 10、100）不是回文数，除非该整数本身就是 0。

## 输入格式
输入仅包含一行一个整数 n，取值范围为 −2^31 ≤ n ≤ 2^31−1（即 32 位有符号整数的范围）。

## 输出格式
输出一行字符串，如果 n 是回文数则输出 "Yes"，否则输出 "No"（注意大小写）。

## 样例

### 样例 1

**输入：**
```
121
```

**输出：**
```
Yes
```

### 样例 2

**输入：**
```
-121
```

**输出：**
```
No
```

## 数据范围
- 时间限制：1 秒
- 内存限制：256MB
- 从左向右

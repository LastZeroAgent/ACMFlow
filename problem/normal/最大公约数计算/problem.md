# 最大公约数计算
 ## 题目描述
给定两个正整数 a 和 b，请计算它们的最大公约数（Greatest Common Divisor, GCD）。最大公约数是指能同时整除这两个数的最大正整数。本题要求使用高效的欧几里得算法完成计算。

## 输入格式
输入包含多组测试用例。每组测试用例占一行，包含两个正整数 a 和 b（1 ≤ a, b ≤ 10^9），两个整数之间用单个空格分隔。

## 输出格式
对于每组测试用例，输出一个整数，表示 a 和 b 的最大公约数。每个结果占一行。

## 样例

### 样例 1

**输入：**
```
6 9
```

**输出：**
```
3
```

### 样例 2

**输入：**
```
12 8
```

**输出：**
```
4
```

## 数据范围
1. 推荐使用欧几里得算法（辗转相除法），该算法的时间复杂度为 O(log(min(a,b)))，能够高效处理大数。
2. 注意处理特殊情况：当其中一个数为另一个数的倍数时，直接返回较小的数。
3. 确保算法不会进入无限循环，每次迭代都要缩小问题规模。

# 题目文件 `PalindromeNumber.md`
## 题目描述
给定一个整数，判断它是否为回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。例如，121 是回文数，而 123 不是。注意：负数不是回文数，因为负号的存在导致正反读不一致。此外，个位数（包括 0）都是回文数。

## 输入格式
第一行包含一个整数 T (1 ≤ T ≤ 10^5)，表示测试用例的数量。
接下来 T 行，每行包含一个整数 n (−10^9 ≤ n ≤ 10^9)。

## 输出格式
对于每个测试用例，输出一行，如果 n 是回文数则输出 "Yes"，否则输出 "No"（不包含引号）。

## 样例

### 样例 1

**输入：**
```
3<br>121<br>-121<br>10
```

**输出：**
```
Yes<br>No<br>No
```

### 样例 2

**输入：**
```
2<br>0<br>12321
```

**输出：**
```
Yes<br>Yes
```

## 数据范围
- 时间限制：1 秒
- 内存限制：256MB
- 从左向右

# ACM竞赛题目-两数之和的存在性判定 ## 题目描述[详细的题目描述]  
给定一个整数数组 `nums` 和一个目标值 `k`，请你判断数组中是否存在**两个不同的元素**，使得它们的和恰好等于 `k`。注意：这两个元素必须是不同的元素（即下标不同，但数值可以相同）。

## 样例

### 样例 1

**输入：**
```
4<br>2 7 11 15<br>9
```

**输出：**
```
YES
```

### 样例 2

**输入：**
```
3<br>1 2 3<br>7
```

**输出：**
```
NO
```

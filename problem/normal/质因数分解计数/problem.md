# 质因数分解计数 
## 题目描述给定一个大于1的正整数n，将其分解为若干个质数的乘积形式，并统计其中**不同**质因数的个数。例如，当n=12时，可分解为2²×3¹，共有2个不同的质因数（2和3）。

## 输入格式
输入包含多组测试数据，每组数据占一行，每行包含一个正整数n（2 ≤ n ≤ 10^6）。

## 输出格式
对于每组测试数据，输出一个整数，表示n的不同质因数的个数。

## 样例

### 样例 1

**输入：**
```
6
```

**输出：**
```
2
```

### 样例 2

**输入：**
```
8
```

**输出：**
```
1
```

## 数据范围
1. 注意处理n本身是质数的情况，此时结果应为1。
2. 需去重统计质因数，即使某个质因数出现多次也只计一次。
3. 建议使用埃氏筛法预处理质数表以提高大数分解效率。
